Todo App

The To-Do App is a simple and intuitive task management application developed using Angular, HTML, and CSS. This app helps users keep track of their daily tasks by allowing them to add, update, and delete tasks easily.

Features

Add Tasks: Users can quickly add new tasks to their to-do list with a clear input field.
Mark as Completed: Tasks can be marked as completed with a single click, helping users keep track of their progress.
Update Tasks: If a task needs modification, users can easily update the task description.
Delete Tasks: Completed or unwanted tasks can be removed from the list, keeping the to-do list clean and organized.
Responsive Design: The app is fully responsive, ensuring a smooth user experience on both desktop and mobile devices.

Software Requirements
  • Angular
  • HTML
  • CSS
